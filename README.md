# 老汪的Financial笔记

> 财务人的实战笔记 | 让数字讲出商业故事

一个现代化的财务博客网站，专注于财务BP、经营分析和财务管理实战经验分享。

## 📸 项目预览

本项目参考了 [https://www.yysuni.com/blog](https://www.yysuni.com/blog) 的优雅设计风格，结合财务专业特色，打造温暖而专业的财务知识分享平台。

## ✨ 设计特点

### 🎨 视觉设计
- **柔和渐变配色**：采用绿色、蓝色、白色系的舒适配色方案
- **卡片式布局**：圆角设计搭配微妙阴影，营造层次感
- **专业图标**：使用Font Awesome图标库，简洁而专业
- **响应式设计**：完美适配桌面、平板和移动设备

### 🏗️ 布局结构
- **左侧导航栏**：包含首页、近期文章、财务工具包、BP社群、关于我等栏目
- **中央主内容区**：欢迎卡片 + 文章列表展示
- **右侧边栏**：实时时钟、交互式日历、社交媒体链接、工具推荐

### 💫 交互体验
- **流畅动画**：卡片悬浮、点击反馈等细腻的交互动画
- **实时更新**：时钟每秒更新，日历支持月份切换
- **键盘快捷键**：支持 Ctrl/Cmd + N 快速创建文章
- **平滑滚动**：锚点跳转使用平滑滚动效果

## 🎯 功能入口说明

### 主要页面路径
- **首页**: `index.html`
  - 展示欢迎卡片和最新文章列表
  - 包含左右两侧导航和工具栏

- **文章详情页**: `articles/article{N}.html`
  - article1.html - 创业融资找FA（8分钟阅读）
  - article2.html - 投资机构邮箱清单（12分钟阅读）
  - article3.html - BP技巧（6分钟阅读）

### 交互功能入口
1. **导航切换**: 点击左侧导航栏任意菜单项
2. **写文章**: 点击左下角绿色"写文章"按钮
3. **阅读文章**: 点击文章卡片或"阅读全文"链接，跳转到详情页
4. **返回首页**: 文章详情页顶部和底部均有返回按钮
5. **日历操作**: 点击日历左右箭头切换月份
6. **社交链接**: 点击右侧"关注我"卡片中的链接
7. **工具访问**: 点击"实用工具"列表中的任一项
8. **文章分享**: 文章详情页底部可分享到微信/微博

### 键盘快捷键
- `Ctrl/Cmd + N`: 快速打开写文章功能
- `Ctrl/Cmd + K`: 聚焦搜索（预留功能）

## 📊 数据存储方案

如需动态数据管理，可使用项目内置的 RESTful Table API：

### 文章表结构
```javascript
TableSchemaUpdate({
  name: 'articles',
  fields: [
    { name: 'id', type: 'text' },
    { name: 'title', type: 'text' },
    { name: 'excerpt', type: 'text' },
    { name: 'content', type: 'rich_text' },
    { name: 'tags', type: 'array' },
    { name: 'read_time', type: 'number' },
    { name: 'published_at', type: 'datetime' },
    { name: 'views', type: 'number' },
    { name: 'likes', type: 'number' }
  ]
});
```

### API调用示例
```javascript
// 获取文章列表
const response = await fetch('tables/articles?page=1&limit=10');
const data = await response.json();

// 创建新文章
await fetch('tables/articles', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({
    title: '文章标题',
    content: 'Markdown内容...',
    tags: ['财务分析']
  })
});
```
