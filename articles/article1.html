创业融资找FA？这20家机构靠谱，但要小心这些坑

创业者找钱，有两条路。

一条是自己拿着BP到处投，效率低，成功率更低。

另一条是找FA帮你对接投资人，专业的事交给专业的人做。

但FA这个行业，水很深。

今天把这个行业的门道说清楚，顺便给你一份靠谱的FA机构清单。

一、FA到底是干什么的

FA，全称Financial Advisor，财务顾问。

说白了，就是帮创业者找钱的中介。

但这个中介，可不是随便谁都能当的。

好的FA能做什么？

  1. 帮你梳理商业逻辑：你的BP可能写得一塌糊涂，FA会帮你重新梳理，让投资人看得懂。
  2. 匹配合适的投资人：不是所有投资人都适合你，FA知道哪些机构在看你这个赛道，哪些机构最近在出手。
  3. 谈判和估值建议：投资条款怎么谈，估值定多少，FA比你有经验。
  4. 加速融资进度：自己找投资人，可能要半年一年，FA能帮你缩短到2-3个月。

差的FA会干什么？

  1. 拿你的BP到处乱发：投资圈就这么大，你的BP被群发一遍，基本就废了。
  2. 收了钱不办事：有些FA收了服务费，转身就消失了。
  3. 把你介绍给不靠谱的投资人：市面上鱼龙混杂的假投资人到处都是，差的FA根本不筛选。
  4. 承诺保证融资成功：这是最大的骗局，没有任何FA能保证你融到钱。

二、头部FA都在干什么

先说一个现实：投资机构和FA和创业项目一样，更迭很快。

前几年很火的FA，今年可能就不行了。今年冒出来的新FA，可能明年就倒闭了。

所以，创业者需要注意筛选。

华兴资本

这是FA行业的天花板，没有之一。

华兴不仅做FA，还有自己的投资基金。他们服务的客户，都是行业头部项目：小米、美团、滴滴、字节跳动。

但华兴的门槛极高，普通创业者基本接触不到。

云岫资本

专注硬科技领域的FA，在半导体、新能源、先进制造领域深耕多年。

如果你做的是硬科技项目，云岫是一个不错的选择。

青桐资本

成立于2014年，专注早期和成长期项目的融资服务。

他们的特点是对接速度快，服务的项目数量多，在消费、科技、医疗领域都有布局。

光源资本

光源资本成立于 2014 年，是一家快速成长、以国际视野服务全球企业的产业投行，致力于通过资本服务与生态赋能，助力产业升级发展。目前业务覆盖私募股权融资顾问、并购重组、资产管理、证券承销、产业发展顾问等领域。

三、选FA的五个标准

不是所有FA都适合你，选错了FA，比不找FA还惨。

1. 看案例，不看吹牛

FA跟你讲得天花乱坠，没用。

看他们过去服务过哪些项目，这些项目融资成功了没有，融了多少钱，投资人是谁。

这些信息，都可以在网上查到。查不到的，基本就是吹牛的。

2. 看专业度，不看态度

有些FA上来就跟你称兄道弟，很热情。

但热情不代表专业。

问他们几个问题：你们在这个赛道服务过几个项目？你们认为我们的估值应该定在什么区间？你们打算给我们对接哪些投资人？

答不上来的，直接pass。

3. 看收费模式，不看承诺

正规的FA，收费模式一般是这样的：

  • 服务费：几万到几十万不等，用来覆盖FA的人力成本。
  • 成功费：融资成功后，按融资金额的3%-5%收取。

如果一个FA跟你说，不收服务费，只收成功费，听起来很美好。

但这种FA往往不靠谱，因为他们没有动力认真服务你，可能同时服务几十个项目，哪个成了算哪个。

更要警惕的是，承诺保证融资成功的FA，99%是骗子。

融资能不能成功，取决于你的项目本身，没有任何FA能保证。

4. 看资源，不看关系

有些FA跟你说，我跟某某投资人关系很好，能帮你直接约到合伙人。

听听就行，别当真。

真正有价值的FA，是能给你提供一份完整的投资人名单，告诉你这些投资人的投资偏好、出手阶段、最近的投资案例。

这才是资源，而不是所谓的关系。

5. 看口碑，不看宣传

FA行业是个圈子，谁靠谱谁不靠谱，创业者之间都会互相分享。

多问问同行，多看看行业交流群里的讨论，别只听FA自己怎么说。

五、几个避坑指南

FA行业水很深，创业者必须擦亮眼睛。

1. 警惕假FA

有些人打着FA的名义，实际上是骗子。

他们的套路是这样的：

  • 先跟你说，我们可以帮你对接投资人，不收服务费。
  • 然后跟你要一笔"尽调费"、"材料费"、"差旅费"。
  • 收了钱之后，要么消失，要么拖着不办事。

正规的FA，只收服务费和成功费，不会有其他名目的费用。

2. 警惕假投资人

有些FA会把你介绍给假投资人。

这些假投资人的套路是：

  • 先表示对你的项目很感兴趣，约你面谈。
  • 谈完之后，跟你说需要先交一笔"投资意向金"、"保证金"。
  • 收了钱之后，要么消失，要么找各种理由不投。

正规的投资人，绝对不会让创业者先交钱。

3. 警惕对赌协议

有些FA会跟你签对赌协议，比如：

  • 如果融资成功，FA收5%的成功费。
  • 如果融资失败，创业者要赔偿FA的服务费。

这种协议，千万别签。

融资能不能成功，不仅取决于FA的努力，还取决于你的项目本身、市场环境、投资人的偏好。

让创业者承担所有风险，FA不承担任何责任，这本身就不合理。

4. 警惕BP泄露

你的BP，是你的核心商业机密。

有些不靠谱的FA，会把你的BP到处群发，甚至卖给你的竞争对手。

所以，在跟FA签约之前，一定要签保密协议。

而且，最好不要一次性把完整的BP给FA，先给一个简版，等确定靠谱了，再给完整版。

5. 警惕过度承诺

有些FA会跟你说：

  • 我保证三个月内帮你融到钱。
  • 我保证给你拿到某某投资人的投资。
  • 我保证帮你融到XXX万。

这些都是过度承诺。

融资有太多不确定性，没有任何FA能保证。

能做出这种承诺的FA，要么是新手不懂行，要么是骗子。

六、自己找投资人 vs 找FA

很多创业者纠结：到底该不该找FA？

我的建议是：看你的项目阶段和融资需求。

什么时候不需要FA？

  1. 天使轮阶段：如果你融的是几十万、一两百万的天使轮，找FA不划算。FA的成功费按比例收，你融100万，给FA 5万，这5万可能就是你一个月的工资成本。
  2. 你有投资人资源：如果你之前创过业，或者在某个行业深耕多年，手里有一堆投资人的联系方式，那就不需要FA了。
  3. 你的项目特别好：如果你的项目数据很亮眼，投资人会主动找上门，这时候也不需要FA。

什么时候需要FA？

  1. A轮及以后：A轮往后，融资金额大了，对接的投资人层级也高了，FA能帮你节省大量时间。
  2. 你没有投资人资源：如果你是第一次创业，不认识投资人，FA能帮你打开局面。
  3. 你想快速融资：如果你的现金流撑不了太久，需要尽快拿到钱，FA能加速融资进度。
  4. 你需要专业建议：估值定多少、投资条款怎么谈，这些都是专业活，FA比你有经验。

七、最后说几句掏心窝的话

FA只是工具，不是救命稻草。

如果你的产品不行，数据不行，商业模式不行，再好的FA也救不了你。

投资人不是傻子，他们要的是回报，不是情怀。

融资的本质，是把未来的钱提前拿到现在用。

但前提是，你得证明你的未来是有价值的。

产品做出来了，用户增长起来了，营收跑出来了，融资自然会来。

产品没做好，就想着融资，本末倒置。

FA能帮你对接投资人，能帮你优化BP，能帮你谈判条款。

但FA改变不了你的产品，改变不了你的团队，改变不了你的商业模式。

这些，只能靠你自己。

所以，与其花时间找FA，不如把产品做到用户离不开。

用户认可你，投资人才会认可你。

这才是融资的底层逻辑。

最后再说一遍：

请寻求正规渠道找投资人，注意不要上当受骗。

市面上鱼龙混杂的投资人和投资机构到处都是，看不懂的协议别签，听不懂的条款别答应，交钱之前多想想。

这份FA机构清单，给你做个参考。

但选不选，怎么选，还得看你自己的判断。
