## 📱 如何分享到微信朋友圈？

```
1. 点击"下载分享卡片" 
   → 图片保存到本地

2. 打开微信朋友圈
   → 选择刚下载的图片

3. 配上文案发布
   → 例如："干货分享 | 扫码阅读完整文章"

4. 朋友扫描二维码
   → 直接访问文章页面
```

## 🔧 遇到问题？

### 问题：二维码不显示
**解决**：清除浏览器缓存，刷新页面

### 问题：下载的图片模糊
**解决**：卡片已是高清（1200x630px），可能是查看软件的问题

### 问题：点击分享没反应
**解决**：打开浏览器控制台（F12），查看是否有错误
