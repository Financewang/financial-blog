## 💻 技术实现

### 核心技术栈

1. **Canvas API** - 图形绘制
   - 渐变背景绘制
   - 文字渲染和自动换行
   - 图片绘制（二维码）
   - 图片导出（toDataURL）

2. **QRCode.js** - 二维码生成
   - 版本：1.0.0
   - 尺寸：300x300px
   - 纠错级别：H（最高）
   - 颜色：黑色二维码 + 白色背景

3. **CSS3 动画** - 交互效果
   - Keyframes 关键帧动画
   - Transform 变换
   - Transition 过渡
   - Backdrop-filter 背景模糊

4. **JavaScript ES6+** - 逻辑控制
   - Class 类封装
   - Async/Await 异步处理
   - Arrow Functions 箭头函数
   - Template Literals 模板字符串

### 代码结构

```javascript
ShareCardGenerator (类)
├── constructor()          // 初始化
├── init()                 // 创建模态框
├── createModal()          // 生成 HTML
├── bindEvents()           // 绑定事件
├── generateCard()         // 生成卡片主流程
├── drawBackground()       // 绘制背景
├── drawBrandInfo()        // 绘制品牌
├── drawArticleInfo()      // 绘制文章
├── drawQRCode()           // 绘制二维码
├── drawDecorations()      // 绘制装饰
├── wrapText()             // 文字换行
├── openModal()            // 打开模态框
├── closeModal()           // 关闭模态框
├── downloadCard()         // 下载卡片
├── copyLink()             // 复制链接
└── showToast()            // 显示提示
```
